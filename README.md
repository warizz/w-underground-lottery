# Underground lottery
Lottery system simulator

[client demo](https://w-underground-lottery.herokuapp.com)  
[server api](https://github.com/warizz/w-underground-lottery-api)

## technology stack
```
client
  react/
    react-redux
    react-router@3

server
  node/
    express
    mongoDB
    mocha
```

## required

